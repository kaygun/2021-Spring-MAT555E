Model Building via Optimization

Fit, Cost and Penalty

  • Larger the fit, larger the similarity, smaller the penalty
  • Larger the cost, smaller the similarity, larger the penalty

 

Distance measures

  • Lp-distance for points in ℝn

    ∑x∥f(x) − g(x)∥p

  • Kullback-Leibler-divergence for distributions

    ∑xp(x)log (p(x)/q(x))

Similarity measures

  • Cosine similarity

    Sim(x, y) = (x ⋅ y)/∥x∥∥y∥

Supervised Learning

Data and model

  • Data set D consists of (x, y)
    • x input
    • y expected output
  • Model is a function f(x, θ)

    Cost(θ) = ∑xdist(f(x, θ), y)

  • Best model is one where Cost is minimized.

Optimization

Gradient descent

  • F: ℝm → ℝ
    • We want argminθF(θ)
    • We find it iteratively

      θn + 1 = θn + γ∇F(θn)

  • Optimal θ gives best fitting model.

Regularization

  • In Regularization add a penalty term into the Cost function

    Cost(θ) = Ψ(θ) + ∑xdist(f(x, θ), y)

  • Occam’s Razor = Optimal θ with minimal penalty

Regression

  • Data: X, y = (x1(i), …, xn(i); y(i))
  • Cost: ∥y − βX∥
  • ℓp-regularization: ∥y − βX∥ + λ∥β∥p
    • p = 1: Lasso
    • p = 2: Ridge
    • p = 1 and p = 2: Elastic Net
